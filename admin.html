Personal Access Token do GitHub
Necessário para salvar dados no repositório e compartilhar com todos os usuários
Como gerar o token:
1. Acesse github.com/settings/tokens → Fine-grained tokens
2. Clique em Generate new token
3. Repository access → nip-globo/orcamento
4. Permissions → Contents → Read and write
5. Copie o token gerado e cole acima
⚠️ O token fica salvo apenas neste navegador. Cada admin precisa configurar o seu.
